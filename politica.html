Política de Privacidade

Última atualização: Junho de 2025

Esta política de privacidade descreve como o aplicativo Makebot lida com as informações dos usuários.

Coleta de dados

O Makebot não coleta, armazena ou compartilha nenhum dado pessoal dos usuários.

Uso da API do WhatsApp

Este aplicativo utiliza a API oficial do WhatsApp (Meta) para envio e recebimento de mensagens automatizadas. Todas as interações são processadas diretamente com os servidores da Meta, e nenhum dado é retido pelo Makebot.

Segurança

Mesmo não coletando dados, nos comprometemos a seguir boas práticas de segurança e privacidade digital para garantir a proteção das informações trafegadas entre sistemas.
